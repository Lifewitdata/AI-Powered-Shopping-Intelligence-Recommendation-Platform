# AI-Powered Shopping Intelligence & Recommendation Platform

## Overview

This project analyzes customer shopping behavior and builds a recommendation system using ecommerce interaction data. The goal is to understand how users move from product views to purchases and use those patterns to generate personalized product suggestions.

The project combines product analytics, customer behavior analysis, SQL insights, and machine learning to simulate workflows used in consumer technology companies.

---

## Business Problem

Online shopping platforms collect large volumes of customer interaction data, but raw data alone does not help users make better decisions.

Key questions addressed:

- Which products are users most likely to purchase?
- Which categories and brands drive the highest activity?
- What factors influence conversion?
- How can user behavior improve recommendations?

---

## Dataset

Source:
E-commerce Behavior Dataset (October–November 2019)

Data includes:

- Product views
- Cart events
- Purchases
- User sessions
- Product categories
- Brand information

Dataset size:

- 2,000+ behavioral events
- 658 users
- October + November shopping activity
